# Dashboard Portfolio Strategy

This directory contains conceptual layouts and interactive strategies for the BI projects.

### 1. Hotel Booking Dashboard
- **Target KPIs:** Occupancy Rate, Cancellation Trend, ADR (Average Daily Rate).
- **Visuals:** Heatmaps for seasonal trends, Slicers for market segments.

### 2. Loan Conversion Analysis
- **Target KPIs:** Conversion Rate, Marketing ROI.
- **Visuals:** Funnel charts for campaign progression, Scatter plots for income-to-conversion correlation.
